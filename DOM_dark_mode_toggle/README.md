# DOM - Dark Mode Toggle

## Description

A simple project where clicking a button toggles the website between light and dark themes.

## What I Learned

- How to select elements from the DOM using `querySelector`).
- How to listen for events using `addEventListener` ('click', callback).
- How to manipulate element classes dynamically with `classList.toggle()`.
- How to combine JavaScript and CSS for dynamic style changes.
- How to update button text based on the current mode using textContent.
