# DOM - Add/Remove List Items

## Description

A simple project where you can dynamically add new list items or remove the last item by clicking buttons.

## What I Learned

- How to select elements from the DOM using `querySelector()`.
- How to listen for events using `addEventListener()` ('click', callback).
- How to create elements dynamically using `createElement()`.
- How to add elements to the DOM with `appendChild()`.
- How to remove elements from the DOM using `removeChild()` and `lastElementChild`.
- How to manage a counter to keep track of dynamic elements.
