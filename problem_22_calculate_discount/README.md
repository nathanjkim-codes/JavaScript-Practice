# Problem: Calculate Discount

## Description

This is a simple JavaScript practice problem.  
The goal is:

1. Take the original price and a discount percentage.
2. Calculate the discount amount.
3. Subtract the discount from the original price.
4. Return the final price.

## Example

```js
calculateDiscount(100, 20); // 80
calculateDiscount(250, 15); // 212.5
calculateDiscount(50, 0); // 50
calculateDiscount(120, 50); // 60
```
