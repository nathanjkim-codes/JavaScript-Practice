# Problem 20 - Shopping Cart Total

## Description

A JavaScript practice problem where you calculate the total price of items in a shopping cart.  
The function takes an array of numbers representing product prices and returns their total sum.  
If the array is empty, it returns 0.

**Example:**

```js
getCartTotal([100, 200, 50]); // 350
getCartTotal([]); // 0
getCartTotal([5, 10, 15]); // 30
getCartTotal([12.5, 7.5, 10]); // 30
```

## What I Learned

- How to use reduce to sum elements in an array.

- How to handle empty arrays safely.

- How to write reusable functions for array calculations.

- How to test JavaScript functions in Node.js
