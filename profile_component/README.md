# Member Profile Card Component

This is a simple **member profile card** component built with **HTML, CSS, and JavaScript**.  
It allows displaying and editing member information dynamically.

---

## Features

- Display member information:
  - Name
  - Role (Parent / Child)
  - DOB (Date of Birth)
  - Gender
- **Edit functionality**:
  - Click "Edit" to dynamically edit name, role, DOB, and gender
  - Role and gender can be selected using dropdowns (in the future / optional)
- **Delete functionality**:
  - Click "Delete" to remove the member card from the DOM

## What I Learned

- How to **structure a component** with HTML, CSS, and JS.

- How to **dynamically create and remove elements** in JavaScript.
