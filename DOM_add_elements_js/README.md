# DOM - Add Elements Using JavaScript

## Description

Dynamically adds elements to a webpage using only JavaScript.

## Features

- Adds a red `<p>` with text: “Hey I’m red!”
- Adds a blue `<h3>` with text: “I’m a blue h3!”
- Adds a pink `<div>` with black border containing:
  - `<h1>`: “I’m in a div”
  - `<p>`: “ME TOO!”

## What I Learned

- Selecting elements with `querySelector()`
- Creating elements with `createElement()`
- Appending elements using `appendChild()`
- Setting text with `textContent`
- Styling elements dynamically with `.style`
- Nesting elements within other elements
