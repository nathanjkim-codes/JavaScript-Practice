# DOM Shopping List

A simple dynamic shopping list built with HTML, CSS, and JavaScript.

---

## Features

- Add items to the shopping list dynamically
- Prevent form submission from refreshing the page
- Clear and focus the input after adding an item
- Basic styling with hover effects for better user experience

---

## Code Highlights

- **Form Handling**: Uses `submit` event with `e.preventDefault()` to prevent page refresh
- **DOM Manipulation**: Creates `li` elements dynamically and appends to the list
- **Styling**: Simple CSS to make the list and form visually appealing

---

## What I Learned

- Handling form submit events
- Creating and appending elements dynamically
- Clearing input
- Basic CSS styling for layout and hover effects
