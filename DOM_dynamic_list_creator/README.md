# DOM - Dynamic List Creator

## Description

A small project that allows users to dynamically add list items to a webpage by typing text and clicking the “Add Item” button.

## What I Learned

- How to create and append elements using `createElement()` and `appendChild()`
- How to handle user input with text fields and buttons
- How to trim and validate input using `.value.trim()`
- How to clear input after use
- How to build dynamic, interactive DOM structures with JavaScript
