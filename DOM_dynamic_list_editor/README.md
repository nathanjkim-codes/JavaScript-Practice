# Dynamic List Editor

## Description

A simple and interactive To-Do list application where users can add items and remove them as needed. Built using pure HTML, CSS, and JavaScript.

## Features

- Add new items to the list dynamically.
- Remove items individually using a "Delete" button.
- Input is trimmed and empty submissions are ignored.
- Clean and minimal styling for readability.

## What I Learned

- Selecting and manipulating DOM elements with `querySelector`.
- Creating, appending, and removing elements dynamically using `createElement`, `appendChild`, and `removeChild`.
- Using event listeners with `addEventListener` for click events.
- Working with inline styling in JavaScript.
- Clearing input fields after adding items.
- Styling HTML elements using CSS to create a simple and elegant UI.

## How to Use

1. Open `index.html` in a browser.
2. Type an item into the input field.
3. Click the "Add" button to add it to the list.
4. Click "Delete" next to any item to remove it.
